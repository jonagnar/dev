# tea — Forgejo/Gitea CLI

Official Gitea `tea` CLI (PRs, issues, repos) for the self-hosted Forgejo at
http://localhost:3000. Binary is re-downloadable (see `tools/manifest.md`).
Source: https://gitea.com/gitea/tea

Auth (one-time):
  tea login add --name forgejo --url http://localhost:3000 --token <FORGEJO_PAT>
